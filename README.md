# Berkah Mebel Ayu - E-Commerce Website

Platform e-commerce untuk penjualan furniture kayu berkualitas premium.

## 📁 Struktur File

```
e-commery/
├── img/                      # Folder untuk gambar produk
├── index.php                 # Halaman utama (landing page)
├── login.php                 # Halaman login
├── register.php              # Halaman registrasi
├── register_process.php      # Proses registrasi
├── auth.php                  # Proses autentikasi login
├── dashboard.php             # Halaman dashboard/belanja
├── cart.php                  # Halaman keranjang belanja
├── profile.php               # Halaman profil pengguna
├── orders.php                # Halaman riwayat pesanan
├── settings.php              # Halaman pengaturan
├── wishlist.php              # Halaman daftar keinginan
├── logout.php                # Proses logout
└── users.json                # Database pengguna (JSON)
```

## 🎨 Desain & Teknologi

- **Frontend:** Tailwind CSS 3
- **Backend:** PHP 7+
- **Icons:** Font Awesome 6.4
- **Fonts:** Playfair Display + Inter
- **Database:** JSON (dapat di-upgrade ke MySQL)

## 🛠️ Instalasi

1. Tempatkan folder di folder `htdocs` XAMPP
2. Akses via browser: `http://localhost/e-commery/`
3. Login dengan akun demo atau buat akun baru

## 📱 Responsive Design

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
